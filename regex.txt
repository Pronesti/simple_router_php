REGEX(regular expressions):
. -> Cualquier caracter
\ -> escape caracter se usa por ejemplo con /
\w -> word
\n -> salto de linea
\
() -> grupos
[] -> conjunto de caracteres posibles
[^abc] -> conjunto de caracteres excluidos
[a-z] [0-9] -> entre caracteres
(a|b) -> a or b
